# STAVBY PRO Release Manager

Windows aplikácia na prípravu a publikovanie vydaní projektu STAVBY PRO.

## Funkcie

- publikovanie STAVBY PRO na GitHub,
- aktualizácia version.json,
- vytvorenie GitHub Release,
- nahrávanie a nahradenie APK,
- zobrazovanie priebehu publikovania,
- automatická kontrola GitHub pripojenia,
- kontrola nových verzií Release Managera.

## Aktualizácie

Najnovšie vydanie je dostupné v časti Releases.

## Inštalácia

1. Stiahni ZIP z najnovšieho GitHub Release.
2. Rozbaľ celý obsah ZIP do samostatného priečinka.
3. Spusti súbor STAVBY_PRO_RELEASE_MANAGER.exe.
